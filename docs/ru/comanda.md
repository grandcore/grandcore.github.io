## Условия для Контрибьюторов

- Для фонда и каждого проекта будет вестись документ (летопись) с описанием вклада каждого участника в каждой вехи разработки.
- Контрибьюторы последний мажорной верскии будут указываться в разделе "команда" на главной странице.
- Участники разработки последней мажорной версии имеют приоритет при выполнении задач на которые выделяются деньги.
- Все Контрибьюторы, внёсшие значительный вклад в запуск платформы, при их желании постоянно учавствовать в жизни фонда, получат статус Хранителя, который даст равный с другими Хранителями голос в принятии всех ключевых решений.
