# Присоединиться к нам

Со времени старта проекта мы пригласили в него более ста тысяч человек. Не смотря на в целом высокие оценки наших идей, нам пока не удалось собрать полноценную команду. То есть, более тысячи человек сказали, что готовы принять участие в проекте, после чего слились. В итоге - нас всего несколько человек. Ищем единомышленников дальше =)

## Условия для Контрибьюторов

- Для фонда и каждого проекта будет вестить документ (летопись) с описанием вклада каждого участника в каждой вехи разработки.
- Контрибьюторы последний мажорной верскии будут указываться в разделе "команда" на главной странице.
- Участники разработки последней мажорной версии имеют приоритет при выполнении задач на которые выделяются деньги.
- Все Контрибьюторы, которые внесут значительный вклад в запуск платформы, при их желании на постоянной основе учавствовать в жизни фонда, получат статус Хранителя, который даст равный с другими Хранителями голос в принятии всех ключевых решений.
