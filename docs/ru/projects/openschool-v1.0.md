# OpenSchool - описание версии 1.0

- Данный текст является черновиком
- Над экранами ведётся активная разработка, текущий статус [смотрите](Хhttps://www.figma.com/file/t8ttKwTUGHXPusH4MVFa31/Share?node-id=7%3A1210) в Фигме
- Обсуждение ведётся в [рабочей группе](https://t.me/openschool_chat) в Телеграме

_Горячо приветствуется любая критика в нашем чате после полного изучения данного документа._

Это первая итерация. Ниже описаны общие концепции и концепции по каждому экрану. Текущие решения по UI предлагаются в качестве примера и они НЕ несут даже рекомендательный характер.

## Основные концепции

- Неавторизованные пользователи имеют доступ ко всей информации на платформе, включая список карточек и рабочим процессам.
- Авторизованные пользователи могут подписываться на курсы, изучать карточки с помощью индивидуальных спринтов и сохранять прогресс своего обучения. Кроме того, иметь голос при принятии решений, участвуя в процессах, например, создавая и дополняя курсы, в рамках прав групп в которых они состоят. 

## Главная страница

![OpenSchool](../../_media/openschool-1-1.png)

- Краткая информация о фонде и промо-видео, ссылка на подробную информацию
- Список всех разделов и самых популярных подразделов. Возможно каждому подразделу имеет смысл добавить иконку для читаемости
- Блоки с новыми и недавно принятыми процессами
- Новые сборы средств участников, которые готовы выполнить большие объёмы работы

## Каталог - Список курсов

![OpenSchool](../../_media/openschool-2-1.png)

- В разделе "Мои курсы" содержится список всех активных и неактивных курсов пользователя
- При нажатии на раздел выводятся все подразделы
- Поиск по разделу фильтрует курсы по ключевому слову
- Дополнительная фильтрация и сортировка находится за одной иконкой.
- Три точки не нужны=)
- Курсы выводятся в строчном режиме с указанием группы (например, Английский язык, уровень А2). В случае, если пользователь изучает данный курс или он находится у него на паузе - выводятся соответствующие значки (плей и пауза, по аналогии с плеером)

## Каталог - Курс

![OpenSchool](../../_media/openschool-2-2.png)

- Для авторизованных пользователей доступен переключатель активности и личная статистика по курсу
- Общая статистика по курсу
- Случайные карточки курса
- Топ редакторов 
- Подборки книг, статей и видео на тему курса
- Текущие сборы средств (если они есть)
- Ссылка на процессы (кстати, блок для них тоже нужен)
